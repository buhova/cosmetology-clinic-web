# 🩺 Cosmetology Clinic Web

[🌐 Live Demo](https://buhova.github.io/cosmetology-clinic-web/)

A responsive website for a cosmetology clinic showcasing services, staff, and consultation booking. Designed to provide clients with a smooth and engaging user experience.

📌 About
Cosmetology Clinic Web allows visitors to:
- Request a callback for consultations
- Explore the "Why Choose Us" section
- Browse available procedures and services
- Locate the clinic on an interactive map
- Navigate seamlessly on desktop and mobile devices

🚀 Technologies Used
- HTML 5
- SCSS / CSS
- JavaScript
- Responsive design for mobile and desktop
